# Effect of intraspecific plant variation on animal sensory behaviour

This R project presents a comprehensive analysis of fruit traits and animal behavior. 
It examines how various fruit characteristics influence animal sensory responses 
across three sensory modalities.


## Files
- `R_script_analysis.Rmd` – main analysis script  
- `raw_table_scent.csv` – raw GCMS data  
- `Species_scent_sugar` - Scent-PCA and sugar content data 
- `spectraldata_species`- raw spectral data 
- `pca_species` - color_PCA 
- `Species_traits` - Hardness, color-PCA and sugar content data
- `Data_mixed_model`- Complete data for GLMM model run 


## Additional information 
- For simplicity we used the local species names in the file naming and in the script. 
- The following scientific names correspond to the respective species names: 
- `Fohanasity - Psychotria reducta`
- `Guave - Psidium cattleianum`
- `Kalafam - Oncostemum nervosum`
- `Kalafana - Oncostemum botryoides`
- `Vandrika - Craspidospermum verticillatum`
- `Voara - Ficus botryoides`
